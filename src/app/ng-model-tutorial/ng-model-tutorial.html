Angular ngModel Tutorial

What is ngModel?

ngModel is a directive in Angular that enables two-way data binding between the template (view) and the component (model). It automatically synchronizes the value of an input element with a property in the component class.

How it works:

  • The [(ngModel)] syntax is called "banana in a box" and represents two-way binding.
  • When the user types in the input, the component property updates.
  • When the component property changes, the input value updates.
  • This eliminates the need for manual event handling for simple form inputs.

Current value: {{ userInput }}

Note: ngModel requires FormsModule to be imported in the component's imports array.

Example usage: [(ngModel)]="propertyName" on input, textarea, or select elements.
